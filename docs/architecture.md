# Архитектура CS2 AI Bot

## 🏗️ Структура системы

```
┌─────────────────────────────────────┐
│        CS2Bot (Main Class)          │
│  Координирует все компоненты        │
└─────────────────────────────────────┘
            ↓
    ┌───────┼───────┐
    ↓       ↓       ↓
┌─────────────────────────────────────┐
│  Navigation   │ Movement  │ DecisionTree
│  (Навигация) │(Движение) │    (AI)
└─────────────────────────────────────┘
    ↓           ↓              ↓
┌─────────────────────────────────────┐
│        GameState (Состояние игры)   │
│  Хранит информацию о здоровье,      │
│  деньгах, позиции, врагах и т.д.    │
└─────────────────────────────────────┘
```

## 📍 1. Navigation (Навигация)

**Файл:** `src/navigation.py`

Отвечает за:
- Знание карты de_mirage
- Поиск пути между позициями (алгоритм A*)
- Управление промежуточными точками (waypoints)

**Ключевые методы:**
- `find_path(start, end)` - Находит оптимальный путь
- `set_target(position)` - Устанавливает целевую позицию
- `get_next_waypoint()` - Возвращает следующую промежуточную точку

**Пример использования:**
```python
nav = Navigation()
path = nav.find_path("T_ramp", "A_site")
# Путь: ["T_ramp", "T_mid", "mid_top", "A_heaven", "A_site"]
```

## 🚶 2. Movement (Движение)

**Файл:** `src/movement.py`

Отвечает за:
- Физику движения бота
- Обновление позиции каждый фрейм
- Избегание препятствий
- Вычисление скорости и направления

**Ключевые методы:**
- `set_target(position)` - Устанавливает целевую позицию
- `update(delta_time)` - Обновляет позицию
- `_avoid_obstacles(direction)` - Обход препятствий

## 🧠 3. DecisionTree (AI Логика)

**Файл:** `src/decision_tree.py`

Отвечает за:
- Анализ текущей ситуации
- Выбор действия на основе приоритетов
- Тактические решения (атака, защита, отступление)

**Приоритеты решений (от высшего к низшему):**
1. STAY_ALIVE - Выжить (критическое здоровье)
2. BUY_WEAPON - Купить оружие
3. PEEK_ENEMY - Встретить врага
4. ATTACK_SITE - Атаковать сайт (для T)
5. DEFEND_SITE - Защищать сайт (для CT)

**Пример дерева решений:**
```
if health < 30:
    → Отступить на безопасную позицию
elif no_weapons and can_buy:
    → Купить оружие
elif enemy_visible:
    → Вступить в бой
elif team == "T":
    → Атаковать сайт
else:
    → Защищать сайт
```

## 💾 4. GameState (Состояние игры)

**Файл:** `src/game_state.py`

Хранит информацию:
- Здоровье и броня
- Деньги
- Текущая позиция
- Список врагов и товарищей
- Состояние раунда
- Имущество (оружие)

## 🗺️ 5. Map Data (Данные карты)

**Файл:** `config/de_mirage_map.py`

Содержит:
- Координаты всех ключевых позиций
- Спавны T и CT
- Бомбосайты A и B
- Препятствия
- Граф навигации

## ⚙️ 6. Configuration (Конфигурация)

**Файл:** `config/bot_config.py`

Настройки:
- Уровень сложности (1-5)
- Скорость движения
- Радиус обнаружения врагов
- Приоритеты целей
- Поведение (агрессивность, осторожность)

## 🔄 Цикл работы бота

```
1. Инициализация
   ├─ Создание всех компонентов
   ├─ Установка начальной позиции
   └─ Запуск основного цикла

2. Основной цикл (каждый фрейм)
   ├─ Обновление движения
   ├─ Обновление позиции
   ├─ Проверка достижения точки маршрута
   └─ Если нужно решение → make_ai_decision()

3. Принятие решения (каждые 0.1 секунды)
   ├─ Анализ состояния игры
   ├─ Выполнение дерева решений
   ├─ Выбор действия с наивысшим приоритетом
   └─ Выполнение действия

4. Действия
   ├─ move: Движение к целевой позиции
   ├─ buy: Покупка оружия
   ├─ attack: Атака на врага
   └─ defend: Защита позиции

5. Выход
   └─ Остановка бота, сохранение логов
```

## 🚀 Расширение функционала

### Добавление новой позиции на карту
1. Добавить координаты в `config/de_mirage_map.py`
2. Добавить соединения в `NAVIGATION_GRAPH`
3. Использовать в `DecisionTree`

### Добавление нового поведения
1. Добавить новый `BotObjective` в `src/decision_tree.py`
2. Реализовать метод `_decide_*`
3. Интегрировать в `make_decision()`

### Интеграция с GSI
1. Настроить GSI сервер в CS2 конфиге
2. Получать данные через `GameState.update_from_gsi()`
3. Использовать актуальные данные в решениях
