# GSI интеграция для CS2 AI Bot

## 🔧 Как настроить GSI в CS2

### Шаг 1: Найди файл конфига CS2

Путь:
```
C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\gamestate_integration_cs2bot.cfg
```

Или для другой установки:
```
<Твой путь к Steam>\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\
```

### Шаг 2: Создай новый файл конфига

Если папка `cfg` не существует - создай её.

Создай файл: **gamestate_integration_cs2bot.cfg**

### Шаг 3: Скопируй конфиг

Содержимое файла `config/cs2_gsi_config.json`:

```json
{
  "uri": "http://127.0.0.1:3000/",
  "timeout": 5.0,
  "buffer": 0,
  "throttle": 0,
  "heartbeat": 2.0,
  "auth": {
    "token": ""
  },
  "data": {
    "map": 1,
    "player_id": 1,
    "player_state": 1,
    "player_weapons": 1,
    "player_match_stats": 1,
    "provider": 1,
    "round": 1,
    "bomb": 1,
    "all_players": 1,
    "phase_countdowns": 1
  }
}
```

### Шаг 4: Запусти бота

```powershell
python src/main.py
```

Выбери режим: **2 - Живое подключение (с GSI)**

### Шаг 5: Запусти игру

```
Запусти CS2 локальный матч или демо
```

---

## ✅ Проверка подключения

Когда бот запущен в режиме GSI:
- Он выведет: **"⏳ Ожидание подключения CS2..."**
- Запусти игру
- Если всё правильно: **"✅ CS2 подключена!"**

---

## 🎯 Что будет работать после подключения:

✅ Бот получит реальные данные из игры:
- Координаты игрока
- Здоровье и броня
- Деньги
- Оружие в инвентаре
- Информация о врагах
- Состояние раунда

✅ Бот сможет:
- Принимать осмысленные решения на основе реальной информации
- Реагировать на врагов
- Адаптироваться к ситуации в игре

---

## 🐛 Если не подключается:

1. **Проверь путь к конфигу** - должен быть в папке `game/csgo/cfg/`
2. **Убедись что конфиг создан** - имя должно начинаться с `gamestate_integration_`
3. **Запусти игру** - конфиг загружается при запуске CS2
4. **Проверь порт 3000** - может быть занят другой программой
5. **Перезапусти бота и игру**

---

## 📝 Доп. информация

GSI (Game State Integration) - это встроенный механизм CS2 для отправки данных о состоянии игры.

**URL для бота:** `http://127.0.0.1:3000/`
**Порт:** 3000
**Локальный сервер:** Работает только на одном ПК
