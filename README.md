# CS2 AI Bot - Autonomous Agent для de_mirage

Автономный AI бот для Counter-Strike 2, который самостоятельно передвигается по карте de_mirage, принимает решения и взаимодействует с игровым окружением.

## 📋 Описание

Этот проект реализует полностью автономного бота с:
- ✅ Системой навигации и понимания карты de_mirage
- ✅ AI логикой для принятия осмысленных решений
- ✅ Системой целей и задач (купля-продажа, позиции, атака)
- ✅ Алгоритмом движения с избеганием препятствий
- ✅ Поведенческими деревьями для сложной логики
- ✅ Интеграцией с Game State Integration (GSI) CS2

## 🧠 Как работает AI

### 1. **Навигация**
Бот знает все ключевые точки на карте de_mirage:
- Спавны (T и CT)
- Бомбосайты (A и B)
- Позиции для защиты
- Пути движения между ними

### 2. **Принятие решений**
Бот использует дерево решений для выбора действия:
```
Начало хода
├─ Проверить здоровье
├─ Проверить деньги
├─ Проверить наличие врагов
├─ Выбрать стратегию (атака/защита)
└─ Выполнить действие
```

### 3. **Движение**
- Вычисляет путь до целевой позиции
- Избегает препятствий и врагов
- Использует естественную траекторию движения

### 4. **Интеграция с игрой**
Бот подключается к CS2 через Game State Integration (GSI) для получения:
- Координат игрока
- Здоровья и брони
- Имущества (оружие, деньги)
- Информации о врагах

## ⚙️ Конфигурация

- Скорости реакции
- Поведения в разных ситуациях
